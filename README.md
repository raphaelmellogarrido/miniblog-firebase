# Mini Blog 🚀

A dynamic and responsive Single Page Application (SPA) built to share posts, stories, and ideas. This project was developed using a modern frontend stack integrated with Firebase for real-time data management and user authentication.

---

## 💻 About the Project

The Mini Blog is a complete web application where users can create accounts, securely log in, and publish posts in real-time. The interface was fully optimized for User Experience (UX), featuring dynamic conditional rendering based on authentication states and seamless navigation.

### Key Features

- **User Authentication:** Secure registration, login, and session management.
- **Dynamic Welcome:** Custom user greeting on the Navbar extracting names or email prefixes dynamically.
- **Real-time Feed:** Instant updates on the homepage whenever a new post is published without needing to refresh.
- **Smooth Navigation:** Internal routing handles client-side transitions instantly.
- **Responsive Layout:** A clean design utilizing a solid Flexbox architecture with a persistent sticky footer.

---

## 🛠️ Technologies & Tools Used

- **Vite** — Next-generation frontend tooling for an ultra-fast development environment.
- **React** — Component-based architecture utilizing hooks (`useState`, `useEffect`) for intelligent state handling.
- **React Router (v6)** — Dynamic single-page routing for seamless view transitions.
- **Firebase Auth** — Secure user session management, login, and registration.
- **Cloud Firestore** — NoSQL database with real-time listeners (`onSnapshot`) for instant data syncing.
- **CSS3 & Flexbox** — Advanced layout positioning, exact alignment (`box-sizing: border-box`), and responsive design.
